そこで現場（営業など）から提案ができないかと思ってました。

今後プロダクトサイクルがもっと短くなって行くと思うので、
ビジネスサイドと技術サイドの間にたてる人がとても重要になってくると思っております。

そのため、私は本質的な課題解決を考え、解決策を考え実行できる人材になりたいと思っています。
